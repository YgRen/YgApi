接口名称：注册

接口测试访问的URL:www.eddywren.com/api/index.php/v1/user/register
接口正式访问的URL: www.jjpicture.com/api/index.php/v1/user/register
请求方式：GET
请求参数

参数 含义 是否必须 说明
username 账号 是 用户账号，用户填写，由11位手机号构成，String
password 密码 是 用户密码，用户填写，String

返回数据类型：JSON

返回参数

参数 含义 说明
Code 状态码 成功操作状态码，成功操作时此次应为0
Message 信息提示 对当前返回内容进行描述
Data 返回的具体数据内容 Data为数据第一级参数，需向下进行具体解析
Data..uid 用户ID 注册成功返回注册用户ID
Data..createtime 注册时间 注册成功返回用户注册时间

错误提示
错误提示由错误码Code和错误信息提示Message组成

返回码 说明
1000 参数错误
1001 注册失败，该注册用户已存在

注：在登录前会进行手机号验证，此接口尚未集成第三方登录，待修正

接口名称：登录

接口测试访问的URL:www.eddywren.com/api/index.php/v1/user/login
接口正式访问的URL: www.jjpicture.com/api/index.php/v1/user/login
请求方式：GET
请求参数

参数 含义 是否必须 说明
username 账号 是 用户账号，用户填写，由11位手机号构成，String
password 密码 是 用户密码，用户填写，String
cid clientid 是 个推第三方推送SDK生产的唯一识别码clientid，开发者实现填写，String，32位
device_name 设备名称 是 用户所使用移动设备名称，开发者实现填写，String
operating_system 操作系统 是 用户所使用的移动设备操作系统名称及版本号，开发者实现填写，String
imei_code IMEI码 是 用户移动设备的IMEI国际唯一识别码，开发者实现填写，String

返回数据类型：JSON

返回参数

参数 含义 说明
Code 状态码 成功操作状态码，成功操作时此次应为0
Message 信息提示 对当前返回内容进行描述
accessToken 令牌 用于用户登录状态保持的令牌
Data 返回的具体数据内容 Data为数据第一级参数，需向下进行具体解析
Data..uid 用户ID 注册成功返回注册用户ID
Data..createtime 用户创建时间 用户注册时间
Data..updatetime 用户更新时间 用户此次登录时间
Data..islock 判断用户是否锁定 默认为0，正常状态，为1时，用户处于锁定状态，不可登录

错误提示
错误提示由错误码Code和错误信息提示Message组成

返回码 说明
1000 参数错误
1002 登录失败，不存在该用户
1003 登录失败，密码错误
1004 登录失败，该用户已在线
